# Porra Mundial 2026

Pagina publica de la porra semanal.

La pagina visible en GitHub Pages es `index.html`. Para actualizarla:

1. Actualiza resultados en `porra-mundial-2026-consolidado.html`.
2. Ejecuta `actualizar_pages.cmd` desde la carpeta principal.
3. Copia `docs/index.html` a este repo o ejecuta `preparar_publicacion.cmd`.
4. Haz commit y push con `.\subir.cmd` desde PowerShell.
